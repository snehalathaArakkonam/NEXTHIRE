# NextHire

NextHire is an AI-powered career tools web app providing ATS resume analysis and professional email generation.

## Features

- ATS resume scoring and breakdown
- Skill gap detection and improvement suggestions
- AI-powered email writer with tone options

## Tech Stack

- React 18
- Vite

## Installation

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Open: http://localhost:5173
